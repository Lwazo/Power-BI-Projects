Business Sales Dashboard (Power BI)

This project presents an interactive Power BI dashboard built using the Business Sales dataset. The goal is to analyze sales performance, customer distribution, and regional trends to support business decision-making.

Data Preparation Cleaned and transformed data using Power Query Removed unnecessary columns (Ship Date, Country) Formatted all fields correctly (dates, numbers, text)

Key Insights Total number of orders per segment Average products sold and sales per region Sales distribution across categories and states Most profitable time periods (Year & Quarter) Most used shipping modes per year Customer distribution by state Category performance across multiple years

KPI Analysis

The dashboard includes KPI visuals to track performance against targets:

Orders: Minimum 100, Target 500 Sales: Minimum $15,000, Target $50,000 Units Sold: Minimum 30, Target 1,000
